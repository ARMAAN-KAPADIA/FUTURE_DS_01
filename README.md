# 📊 Superstore Sales & Profitability Dashboard

## 📌 Project Overview
This project analyzes business sales data to uncover insights related to revenue, profitability, customer segments, and regional performance.

The dashboard is built using Power BI and focuses on delivering actionable business insights.

---

## 🎯 Objectives
- Analyze sales trends over time
- Identify top-performing categories and products
- Evaluate regional performance
- Understand the impact of discounts on profitability
- Provide actionable business recommendations

---

## 🛠️ Tools & Technologies
- Power BI
- Excel / CSV Dataset
- DAX (Data Analysis Expressions)

---

## 📊 Key Insights

- Total Sales: $2.3M+
- Total Profit: $286K+
- Profit Margin: ~12.5%

### 🔹 Business Insights:
- Technology is the most profitable category
- Furniture has high sales but low profitability
- West region is the top-performing region
- High discounts negatively impact profit margins
- Some products generate high sales but low profit

---

## 📈 Dashboard Features
- KPI Cards (Sales, Profit, Orders, Margin)
- Sales Trend Analysis
- Category & Regional Performance
- Discount vs Profit Analysis
- Product-Level Insights
- Interactive Filters (Region, Segment, Category, Ship Mode)

---

## 📂 Files Included
- Power BI Dashboard (.pbix)
- Dataset (CSV/Excel)
